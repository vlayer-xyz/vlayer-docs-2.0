# vlayer Glossary

| Term                        | Definition |
|-----------------------------|------------|
| **Web Proof**               | A cryptographic attestation that specific web data was fetched correctly and untampered. |
| **Vouch**                   | The official browser extension used to generate client-side web proofs from authenticated sessions (cookies, logins, etc.). |
| **MPC Mode**                | Multi-Party Computation mode. Offers stronger privacy guarantees. |
| **Proxy Mode**              | Faster mode where the server acts as a proxy (default for most server-side use). |
| **Notary / TLSNotary**      | The trusted third party that cryptographically attests to the TLS connection. |
| **Journal**                 | The output data from a RISC Zero guest program containing the verified computation result. |
| **vgas**                    | vlayer's gas unit for measuring and billing ZK proof computation resources. |
| **RISC Zero**               | The zero-knowledge VM used to turn web proofs into succinct, on-chain verifiable ZK proofs. |
| **Fake Mode**               | Development-only mode that returns mock proofs instantly without network calls. |
| **Selective Disclosure**    | Proving only specific parts of data while keeping the rest private. |
